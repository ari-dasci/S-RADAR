===================

Robust Anomaly Detection And Recognition (RADAR) is an anomaly detection platform designed to unify and integrate diverse approaches and libraries from the 
literature, along with innovative model variants. The goal is to provide a flexible and extensible framework that ranges from classical methods to advanced 
techniques based on Transformer architectures, also including support for Federated Learning in distributed scenarios.
